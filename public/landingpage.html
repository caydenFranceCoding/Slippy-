Frequently Asked Questions

Is Slippy free to use?

Yes, Slippy's core features are completely free to use. For advanced AI capabilities, you'll need to configure your own Groq API key.

How do I set up the AI features?

To use the AI features, you'll need a Groq API key. Click the settings button in the editor, and enter your API key in the settings panel.

What languages does Slippy support?

Currently, Slippy supports HTML, CSS, and JavaScript, with more languages planned for future updates.

Can I save my work?

Yes, you can save your code to your computer directly from the editor. Future versions will include cloud saving capabilities.

Does Slippy work offline?

The core editor works offline, but the AI features require an internet connection to access the Groq API.

Is my code secure?

Your code stays in your browser and is not stored on any servers. When using AI features, code is sent securely to the Groq API for processing.
